Документация

Для работы с Normalfaggotbot, необходимо:
1. Изначально добавить Normalfaggotbot на ваш дискорд сервер
2. Пригласить бота в голосовой чат, воспользовавшись командой !join
3. Наслаждаться музыкой

Команды

Префикс для команд "!". Весь список команд !help:
1. !add_song - Добавление youtube клипа в очередь
2. !disconnect - Отключения из голосового чата
3. !join - Присоединение в голосовой чат
4. !pause - Приостановить youtube клип
5. !play - Запуск youtube клипа по ссылке
6. !resume - Возобновить youtube клип
7. !skip - Пропустить youtube клип
